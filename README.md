# Загальний огляд проекту
Проект являє собою онлайн-платформу для створення, публікації та проходження навчальних курсів.
Він дозволяє:
- Викладачам створювати курси з модульною структурою, додавати уроки та контент (текст, зображення, відео).
- Створювати теоретичні та практичні завдання для курсів.
- Верифікувати курси адміністраторами перед публікацією.
- Студентам переглядати доступні курси, проходити уроки та виконувати практичні завдання.
- Підраховувати результати та формувати сертифікати після успішного завершення курсу.
Мета проекту – надати зручний інструмент для навчання та управління курсами в університетській або корпоративній освіті.

# Функціональності
## Браузинг курсів
- Пошук курсів по назві та автору.
- Пагінація: обмежена кількість курсів на сторінку (10).
- Перехід на сторінку запису на курс (Enrollment) для перегляду детальної інформації та приєднання.

## Проходження курсу студентом

### Enrollment сторінка:
- Кнопка «Приєднатися» до курсу.
- Інформація про викладачів із посиланнями на їх профілі.
- Загальний опис курсу.

### Сторінка курсу:
- Список модулів курсу.
- Модулі містять теоретичні та практичні завдання:
#### Теоретичні: текст, зображення, відео.
#### Практичні: тести з однією або кількома правильними відповідями, завдання з введенням тексту.
- Сторінка оцінювання (Grading) для практичних завдань.
- Після завершення всіх завдань з’являється кнопка для формування сертифікату.

## Налаштування курсу викладачем
- Викладач може створювати структуру курсу: модулі, уроки, контент, практичні завдання.
- Додати загальний опис курсу.
- Взаємодія з курсом для студентів: теоретичний та практичний контент.
- Після завершення налаштування курс відправляється на верифікацію.

## Верифікація курсів адміністратором
- Адміністратор переглядає список усіх заявок на курси.
- Можливість перегляду деталей курсу.
- Прийняття або відхилення заявки.
- Після затвердження курс стає доступним студентам для проходження.
